TCP/IP je sada protokolů, která slouží ke komunikaci mezi zařízeními na síti.
Jmenuje se podle dvou důležitých protokolů - TCP a IP.

Vrstvy TCP/IP jsou:
- Aplikační
- Transportní
- Síťová
- Vrstva síťového rozhraní

Základní protokoly:
- IP
	- IPv4
	- IPv6
- ARP
- ICMP
- TCP
- UDP

Základní služby:
- DNS
- DHCP
- FTP
- TFTP
- HTTP/HTTPS
- SMTP
- Telnet
- SSH
